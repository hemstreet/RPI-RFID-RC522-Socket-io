Raspberry pi RFID-RC522 RFID reader hooked up with python communicating to a 
node server via socket-io

(Setup Guide)[https://hemstreet.io/raspberry-pi-rc522-setup/]

Installation
============
On the server
`npm install`

On the Pi
`pip install -u socketIO-client`

Setup
=====

On the server
`node server.js`

On the Pi
`python Socket.io`
